# Todo Manager

A simple and interactive **Todo List / Task Manager** web application built with HTML, CSS, and JavaScript.  

---

## Demo

You can see the live project here:  
[View Todo Manager Online](https://darkeagle0912.github.io/todo-manager/)

---

## Features

- Add new tasks with a simple input
- Mark tasks as completed
- Delete tasks
- Filter tasks by:
  - All
  - Active
  - Completed
- Persistent storage using **localStorage**
- Responsive design with **Bootstrap 5**
